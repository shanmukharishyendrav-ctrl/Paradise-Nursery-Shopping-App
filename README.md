# Paradise Nursery Shopping App

## Project Overview

A React-based e-commerce application for an online plant shop called Paradise Nursery. This application allows customers to browse a variety of houseplants, view detailed product information, add items to their shopping cart, and manage their purchases.

## Features

### Product Listing
- Display of at least 6 unique houseplants per category
- Plants organized into multiple categories (Indoor, Outdoor, Succulents, etc.)
- Each plant showcases:
  - Thumbnail image
  - Plant name
  - Price
  - Add to Cart button

### Shopping Cart Functionality
- Add plants to cart with disabled button after addition
- Dynamic cart icon showing total item count
- Cart page displaying:
  - Individual plant thumbnails, names, and unit prices
  - Total cost for each plant
  - Quantity adjustment buttons (increase/decrease)
  - Delete button to remove items
  - Total cart amount
  - Checkout button (Coming Soon)
  - Continue Shopping button

### Navigation
- Navigation bar on all pages
- Links to Home, Plants, and Cart
- Responsive design for different screen sizes

## Technologies Used

- **Frontend Framework**: React
- **State Management**: Redux with Redux Toolkit
- **Styling**: CSS
- **Version Control**: Git

## Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
5. Open your browser and navigate to `http://localhost:3000`

## File Structure

- `App.jsx`: Main landing page component
- `App.css`: Styling for the application
- `AboutUs.jsx`: Company information page
- `ProductList.jsx`: Product listing page component
- `CartItem.jsx`: Shopping cart display component
- `CartSlice.jsx`: Redux slice for cart state management
